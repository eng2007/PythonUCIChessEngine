# Быстрый старт

## Что было добавлено

### ✅ Рокировка
- Короткая рокировка (O-O): король перемещается на 2 клетки вправо
- Длинная рокировка (O-O-O): король перемещается на 2 клетки влево
- Автоматическая проверка всех условий для рокировки

### ✅ Взятие на проходе (En Passant)
- Автоматическое отслеживание возможности взятия на проходе
- Корректное удаление взятой пешки
- Полностью интегрировано в систему ходов

### ✅ UCI Протокол
- Полная поддержка Universal Chess Interface
- Совместимость с Arena, Cutechess, PyChess и другими
- Настраиваемая глубина анализа через UCI команды

## Запуск

### 1. Графический режим (игра с человеком)
```bash
python chess_game.py
```
Выберите сложность 1-4 и играйте мышкой!

### 2. UCI режим (для шахматных программ)
```bash
python uci_interface.py
```

### 3. Тестирование UCI
```bash
# Автоматический тест
python test_uci.py

# Интерактивный режим
python test_uci.py interactive
```

## Примеры UCI команд

```
# Инициализация
uci
isready

# Новая игра
ucinewgame

# Стартовая позиция
position startpos

# Позиция с ходами
position startpos moves e2e4 e7e5 g1f3 b8c6

# Найти лучший ход
go depth 3

# Настроить глубину
setoption name Depth value 4

# Выход
quit
```

## Проверка новых возможностей

### Рокировка
1. Запустите игру
2. Освободите пространство между королем и ладьей
3. Выберите короля - увидите возможность рокировки подсвеченной
4. Кликните на соответствующую клетку для рокировки

### Взятие на проходе
1. Передвиньте свою пешку на 5-ю горизонталь
2. Если противник ходит пешкой на 2 клетки вперед рядом с вашей
3. Вы сможете взять её на проходе на следующем ходу

## Подключение к Arena Chess

1. Скачайте Arena Chess GUI с http://www.playwitharena.de
2. Запустите Arena
3. Engines → Install New Engine
4. Укажите путь к `uci_interface.py`
5. В настройках команды укажите: `python uci_interface.py`
6. Готово! Движок появится в списке

## Подключение к Cutechess

```bash
cutechess-cli \
  -engine cmd=python args="uci_interface.py" name=PythonChess \
  -engine cmd=stockfish \
  -each proto=uci tc=40/60 \
  -games 10 -repeat -pgnout games.pgn
```

## Полезные команды для отладки

В UCI интерфейсе есть нестандартная команда `display` (или `d`):
```
position startpos moves e2e4
display
```
Это покажет текущую позицию в консоли.

## Производительность UCI

- Depth 1: ~0.1 сек на ход
- Depth 2: ~0.5 сек на ход  
- Depth 3: ~2 сек на ход
- Depth 4: ~5-10 сек на ход
- Depth 5: ~30-60 сек на ход

Рекомендуется depth 2-3 для комфортной игры.
